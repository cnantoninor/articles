# pmacros macro artifacts (Article 4)

## Section 1: What this directory is

This directory holds the canonical artifacts for two pmacros default macros proposed alongside Article 4 of the epistemic debt series. Each artifact contains the macro's purpose, full body text (the prompt pmacros injects when the tag appears), one worked example, and prior-art grounding. They are intended to be (a) reusable verbatim by the article, (b) shippable as default macros when pmacros is published.

## Section 2: Index

| Macro | Purpose | File | Prior art |
| --- | --- | --- | --- |
| `<asin>` | Assumption + intent surfacing | asin.md | Chain-of-Verification (Dhuliawala 2023) |
| `<soltree>` | Solution tree exploration | soltree.md | Tree of Thoughts (Yao 2023) |

## Section 3: Status

Both macros are proposed as default pmacros macros and are not yet shipped in the upstream pmacros project. The article hedges accordingly per the fact-check report (see `topics/epistemic_debt/artifacts/articles/article-4-fact-check.md` row M4).
